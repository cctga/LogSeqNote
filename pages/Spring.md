alias:: Spring Framework

- Spring 优势
	- 解耦，简化（单例，文件解析）
	- AOP
	- 声明式事务支持 `@Transactional`
	- 方便测试
	- 集成优秀框架
	- 降低 JavaEE API 的使用难度
	- 优秀的源码
	- 从 Spring 3.0 开始就可以使用 JavaConfig，而不需要 applicationContext.xml 配置
- 不同版本要求的 jdk
	- 5.x -> jdk8+
	- 4.x -> jdk6+
	- 3.x -> jdk5+
